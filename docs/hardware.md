# Hardware setup

### Parts

- S0100/101 arms. We removed the gears from the leader arm to improve more precision when teleoperating.
- 3D printed magnetic chess set from [Chojins/LeRobot-SO-100-Models](https://github.com/Chojins/LeRobot-S0-100-Models)
- Custom arm pedestal for both leader and follower arms from [Chojins/LeRobot-SO-100-Models](https://github.com/Chojins/LeRobot-S0-100-Models): [file](stl/pedestal.stl)
- Soft fin gripper from [XLeRobot](https://github.com/Vector-Wangel/XLeRobot/blob/main/hardware/SO101_soft_fin.stl)
- Custom motor holder + [wire mount](https://www.printables.com/model/1083330-electric-wire-mount-for-miniature-railway-fixation/files) to hold the gripper camera: [mount top](stl/mount_top.stl) and [holder and wire mount](stl/holder_wire_mount.stl)
- Support for the leader gripper: [file](stl/leader_support.stl)

### Cameras

- Top camera, fixed: Razer Kiyo X
- Ring light with tripod
- Gripper camera: 32x32 camera module
